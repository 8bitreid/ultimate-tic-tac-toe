### Getting Started
.NET Core version [2.1.4](https://www.microsoft.com/net/download)  
This repo will follow the github suggested branching.  Please make your Pull Request onto the master branch.  
App creation  
* project creation 
```
dotnet new angular
```
* in project directory run 
```
npm i
```
* Then
```
dotnet run
```
